Privacy Policy

CodeSimplify - AI Code Explainer Extension

Last updated: July 30, 2024

1. Information We Collect

CodeSimplify is designed with privacy in mind. We collect minimal information:

  • Selected Code Text: When you use the extension to explain code, we temporarily process the text you select on GitHub pages.
  • No Personal Information: We do not collect, store, or transmit any personal information, browsing history, or other sensitive data.

2. How We Use Your Information

The selected code text is used solely to:

  • Generate AI-powered explanations of the code
  • Provide you with the requested explanation service

We do not store, analyze, or use this information for any other purpose.

3. Data Storage and Security

Important: All code processing happens in real-time. We do not store any selected code text on our servers or in any database.
  • Selected code is sent to our AI service for processing
  • No data is stored or logged on our servers
  • All communications are encrypted using HTTPS

4. Third-Party Services

We use the following third-party services:

  • Google Gemini AI: For generating code explanations
  • Render.com: For hosting our backend service

These services have their own privacy policies, and we recommend reviewing them.

5. Your Rights

You have the right to:

  • Use the extension without providing any personal information
  • Stop using the extension at any time
  • Contact us with any privacy concerns

6. Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes by updating the "Last updated" date at the top of this policy.

7. Contact Us

If you have any questions about this privacy policy or our data practices, please contact us:

  • Email: privacy@codesimplify.com
  • Website: https://codesimplify.com

8. Consent

By using the CodeSimplify extension, you consent to the collection and use of information as described in this privacy policy.
